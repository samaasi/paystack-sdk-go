## [0.8.0] - 2026-01-20

### Added
- Comprehensive Go SDK for Paystack API.
- Support for all major Paystack services:
  - Transactions, Transfers, Customers, Plans, Subscriptions, Refunds, etc.
- Context-aware methods with timeout and cancellation support.
- Idempotency key support via context.
- Configurable retry logic with exponential backoff.
- GitHub Actions CI/CD workflow for automated testing.
- Comprehensive documentation (`README.md`, `doc.go`).
- Security policy (`SECURITY.md`).
- MIT License (`LICENSE`).
