## Reporting a Vulnerability

We take the security of this SDK seriously. If you discover a security vulnerability, please report it privately.

### **Do not open a public issue.**

Instead, please report it via email.

1. Email your findings to Benson Samaasi at **dev.bensonsamaasi@gmail.com**.
2. Do not disclose the vulnerability publicly until it has been addressed.
3. We will acknowledge your report and work to resolve the issue promptly.
